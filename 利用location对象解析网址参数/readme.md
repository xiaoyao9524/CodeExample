# 利用location对象解析网址参数

### 示例网址：http://www.abcd.com:5000/a/b?a=1&b=2#c=3&d=4

- location.href
      `"http://www.abcd.com:5000/a/b?a=1&b=2#c=3&d=4"`
- location.protocol
      `"http:"`
- location.hostname
      `"www.abcd.com"`
- location.host
      `"www.abcd.com:5000"`
- location.port
      `"5000"`
- location.pathname
      `"/a/b"`
- location.search
      `"?a=1&b=2"`
- location.hash
      `"#c=3&d=4"`
